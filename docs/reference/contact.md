Stay connected with the Charmed Apache Cassandra team

Contact Canonical Data Platform team in
[Matrix](https://matrix.to/#/#charmhub-data-platform:ubuntu.com).

Report your ideas/issues on GitHub:

- Charm issues: <https://github.com/canonical/cassandra-operator/issues>
- Snap packaging: <https://github.com/canonical/charmed-cassandra-snap/issues>

When filing an issue, please include:

- Charm / packaging artifact version
- Juju version & substrate (cloud / VM / LXD / Kubernetes)
- Reproduction steps & expected vs actual behavior
- Logs or relevant command output (`juju status`, `juju debug-log`) where possible
